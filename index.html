🌟 StardustEngine

NFT Gaming Platform pe MultiversX

Platformă blockchain reinventată pentru dezvoltatori și gameri cu NFT-uri, contracte inteligente, asset-uri interoperabile și UI holografic.

MultiversX React + Next.js Rust Smart Contracts NFT Gaming Framer Motion
📦 Repository GitHub 📚 Documentație MultiversX 🚀 Demo local

🎮 Caracteristici Principale

⚡ Smart Contracts (Rust)

Contracte inteligente optimizate pentru gaming cu 13,000+ linii de cod și funcționalități complete pentru NFT-uri.

🎯 NFT Asset System

Sistem complet de asset-uri cu rarități (Common → Legendary), arme, personaje și skin-uri interoperabile.

📈 Player Progression

Sistem de progresie cu XP, level-uri, achievements și statistici on-chain pentru fiecare jucător.

💻 Frontend Modern

React + Next.js 14 cu Framer Motion pentru animații spectaculoase și UI holografic interactiv.

🔗 Integrare MultiversX

Conectare seamless cu wallet-urile MultiversX și interacțiune directă cu blockchain-ul.

🏆 Tournament & Trading

Platformă de turnee competitive cu prize pools în EGLD și sistem de trading pentru asset-uri.

🚀 Cum începi?

Clonează repository-ul

git clone https://github.com/Gzeu/StardustEngine.git

Instalează dependențele frontend

cd frontend && npm install

Rulează server local

npm run dev

Configurează environment

Configurează .env cu adresa contractului și network-ul MultiversX dorit.
